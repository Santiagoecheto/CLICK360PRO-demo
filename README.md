# CLICK360 PRO - Demo Web

Este repositorio contiene una demo de landing page desarrollada como ejemplo para negocios y profesionales.

## Objetivo

Mostrar una estructura base de sitio web lista para adaptar a distintos rubros como:

- Peluquerías
- Psicólogos
- Dentistas
- Comercios
- Servicios profesionales

## Características

- Diseño moderno y responsive
- Sección de servicios dinámica (JSON editable)
- Botón de contacto por WhatsApp
- Estructura adaptable a cualquier cliente

## Personalización

Todos los textos, imágenes, precios y servicios pueden modificarse fácilmente según el cliente.

## Uso

Este proyecto se utiliza como base para crear páginas web personalizadas para clientes reales.
